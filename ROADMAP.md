## Третья, четвёртая и пятая игры + полировка (сверх исходного плана)

- [x] «Жук» (посадка модуля) — третья игра end-to-end, тот же реестр в
      `server/games.js`, без изменений схемы БД (сложность не нужна)
- [x] Уровни сложности в «Кроте» (лёгкая/средняя/сложная) с масштабируемыми
      очками и множителем победы, убывающим со временем
- [x] Цепочка уровней в «Жуке»: победа переносит на следующий уровень
      (топливо −4% и множитель −0.05 за уровень, пол 0.2), счёт копится до
      первой аварии — анти-чит на сервере считает сумму по числу уровней
- [x] «Кабан» (толкай ящики) — четвёртая игра, 4 уровня, очки за
      эффективность (манхэттенское расстояние «ящик → цель» как безопасный
      минимум ходов для анти-чита)
- [x] «Хомяк» (пасьянс, классический Клондайк) — пятая игра, карты как
      DOM-элементы вместо canvas (проще и надёжнее для кликов/тача), очки —
      карты на фундаменте × 10, победа = 1000
- [x] Тематические фоны и мини-анимации геймплея на карточках хаба (все 5 игр)
- [x] Сворачиваемая таблица лидеров, вспышка красным при проигрыше
- [x] Шуточные тексты по итогам партии (победа/поражение), с разбивкой по
      диапазонам счёта/результата, для всех игр, где есть исход победа/поражение
